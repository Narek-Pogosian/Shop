**Shop** is a modern, full-featured eCommerce application built using Next.js. It allows users to browse products, apply various filters, sort by different criteria, and toggle between different product views. The project also includes a client-side cart system, enabling users to add and manage their cart items.

## âœ¨ Features

### **Product Filters**

- **Category Filter**: Browse products by category.
- **Price Filter**: Narrow down results based on price range.
- **Tags Filter**: Filter products by tags such as "Outerwear", "Unisex", etc.
- **Product Attributes**: Filter products by attributes such as size, color, etc.

### **Product Sorting Options**

- Sort products by various criteria including price (low to high, high to low) or latest release.

### **Grid View Toggle**

- Toggle between:
  - **Multi-Column Grid View**: Displays product image, name, and price.
  - **Single Column View**: Shows detailed information about each product, such as description and tags.
- The toggled view is persistent with the help of cookies and works well with SSR.

### **Client-Side Cart System**

- Add, remove, and view products in your cart.
- View a summary of your cart contents with the total price.
- Persistent cart system (cart persists even if the user refreshes the page).
